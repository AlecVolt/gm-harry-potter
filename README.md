# gm-harry-potter

Link: https://alecvolt.github.io/gm-harry-potter/

This project is a responsive web application that displays characters from the Harry Potter universe. The character data is fetched from a public API. The application also uses URL query parameters (e.g., `?param=house&param2=gryffindor`) to filter characters by house. The project is built with HTML, SCSS, and JavaScript, with a responsive design that provides a seamless experience across different devices.
